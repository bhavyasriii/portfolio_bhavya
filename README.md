# 🌐 Bhavyasri Mudireddy | Portfolio

Welcome to my personal portfolio website!  
This is where I bring together my journey, skills, and projects in **Software Engineering** and **Machine Learning**.  



##  What’s Inside?

- 🎯 **About Me** – My story, values, and vision for the future.  
- 🛠 **Skills** – A snapshot of the tech stack I’ve been working with.  
- 🚀 **Projects** – Hands-on work including ML apps, AI assistants, and full-stack builds.  
- 📝 **Blog** – A space where I share learnings, experiences, and reflections.  
- 📬 **Contact** – Quick ways to get in touch or connect on LinkedIn.  



##  Tech Stack

- **Frontend:** React, TailwindCSS  
- **Backend / Tools:** Node.js, Git, AWS  
- **ML / Data Science:** Python, scikit-learn, TensorFlow, Pandas  


##  Features

- Smooth animations and modern UI (developer-style).  
- Dark/Light base styling with gradients and effects.  
- Responsive design (works across devices).  
- Blog section with multiple entries.  
- Easy navigation with smooth scrolling.

##  Getting Started 

If you’d like to run this project locally:

```bash
# Clone the repo
git clone https://github.com/bhavyasriii/my-portfolio.git

# Navigate into project folder
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
